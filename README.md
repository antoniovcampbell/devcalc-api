# DevCalc API com Javalin

API REST para operações matemáticas simples, desenvolvida em Java 21 com Javalin.

## Objetivo
Esta aplicação serve como base para implementação de pipelines de CI/CD com GitHub Actions, incluindo:
- Verificação de código
- Testes automatizados
- Build e empacotamento
- Simulação de deploy

## Tecnologias
- Java 21
- Javalin 6.1.3 (framework web leve)
- JUnit 5 (testes)
- Maven (build)
- GitHub Actions (CI/CD)

## Como executar localmente

### Pré-requisitos
- Java 21+
- Maven 3.8+

1. Clone o repositório:
